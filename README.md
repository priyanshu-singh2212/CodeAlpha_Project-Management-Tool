# 🚀 Project Management Tool (MERN Stack)

A full-stack collaborative Project Management Tool inspired by Trello and Asana.  
It allows teams to manage projects, assign tasks, communicate, and track progress in real time.

---

## 📌 Features

### 👥 User & Auth System
- User Registration & Login (JWT Authentication)
- Secure protected routes

---

### 📁 Project Management
- Create group projects
- Add and manage project boards
- Organize tasks under each project

---

### ✅ Task Management
- Create, update, delete tasks
- Assign tasks to users
- Set due dates
- Mark tasks as completed
- Track overdue tasks automatically

---

### 📊 Dashboard & Stats
- Overview dashboard for all projects
- Task statistics:
  - Todo tasks
  - In Progress tasks
  - Review tasks
  - Completed tasks
- Project progress tracking
- Quick insights for productivity
- User-wise task distribution 

---

### 💬 Collaboration
- Comment system inside tasks
- Team communication within projects
- Activity log for tracking all changes

---

### ⏰ Smart Features
- Due date tracking
- Overdue task detection
- Activity history log

---

### ⚡ Real-Time Features
- Real-time updates using WebSockets (Socket.io)
- Instant task updates across users
- Live comments and notifications

---

## ⚙️ Installation & Setup

### Clone Repo
```bash
git clone https://github.com/your-username/project-management-tool.git
cd project-management-tool
```

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
